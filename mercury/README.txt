== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New Theme' and 'Upload Theme' button.
2. Select the mercury-3.9.10.1.zip file on your computer that you downloaded earlier in the Downloads section of your account on the website https://themeforest.net/downloads
3. Click on the 'Activate' button to use your new theme right away.
4. Go to Appearance > Import Demo Data to install one of the demos.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
